## 리소스 및 참고사항

### 개발 환경
- **OS**: macOS (Apple Silicon)
- **메모리**: 24GB (M5 Pro)
- **Node.js**: v22 이상
- **pnpm**: 최신 버전

### 주요 파일 위치
- 개선 추적: `IMPROVEMENT_TRACKER.md`
- 프로젝트 설정: `PROJECT_SETUP.md`
- Rapid-MLX 설정: `RAPID_MLX_SETUP.md`
- 통합 가이드: `SHARED_ENGINE_INTEGRATION.md`

### 관련 명령어
```bash
# 프로젝트 빌드
pnpm install
pnpm build:packages

# 검증 실행
./scripts/verify-all.sh

# 개발 서버 시작
pnpm dev

# Rapid-MLX 서버 시작 (설치 후)
rapid-mlx serve --model qwen3.5-9b-4bit
```

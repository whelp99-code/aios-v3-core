## Lessons Learned

1. **Pre-flight checks**: Run `pnpm lint && pnpm typecheck && pnpm test && pnpm build` before any release
2. **Type safety**: Never use `unknown` in public interfaces without proper typing
3. **Integration testing**: Test with real dependencies, not just mocks
4. **CI gates**: All CI checks must pass before merge
